# mikenizinski.com — Portfolio Rebuild

Next.js 16 + Tailwind v4 + Sanity v3 + Vercel portfolio site for Mike Nizinski.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **CMS:** Sanity v3
- **Animations:** Framer Motion
- **Hosting:** Vercel

## Local Development

```bash
# Install dependencies
npm install

# Run Next.js dev server
npm run dev        # → http://localhost:3000

# Run Sanity Studio
npx sanity dev     # → http://localhost:3333
```
