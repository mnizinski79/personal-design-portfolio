## Start Session

Kick off a new working session. Do the following steps in order:

1. Run `git fetch && git pull` to sync with the remote
2. Look for and read any of the following documents if they exist — check both the project root and the parent directory, as docs are sometimes kept one level up:
   - `SESSIONS.md` or `SESSION.md` — developer diary / session notes
   - Any `*-PRD.md`, `*-TechSpec.md`, or `*-Spec.md` files
   - `README.md` — project overview
3. Greet Mike with a 2–3 sentence summary of where the project left off and what the immediate next step is, based on what you found
